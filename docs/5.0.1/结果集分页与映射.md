# 结果集、分页与映射（`result`）

## `StatementExecuteResult`

类：`io.github.sinri.keel.integration.mysql.result.StatementExecuteResult`。

包装 Vert.x **`RowSet<Row>`**，提供：

| 方法                                                        | 说明                                                      |
|-----------------------------------------------------------|---------------------------------------------------------|
| `getRowSet()`                                             | 原始 **`RowSet`**                                         |
| `getTotalFetchedRows()`                                   | **`rowSet.size()`**                                     |
| `getTotalAffectedRows()`                                  | **`rowSet.rowCount()`**                                 |
| `getLastInsertedID()`                                     | MySQL 属性 **`LAST_INSERTED_ID`**                         |
| `iterator()`                                              | 遍历 **`Row`**                                            |
| `toMatrix()`                                              | **`ResultMatrix<SimpleResultRow>`**                     |
| `toMatrix(Class<R>)` / `toMatrix(Function<JsonObject,R>)` | **`ResultMatrix<R>`**，需 **`R` 带 `JsonObject` 构造**或自定义映射 |

## `ResultMatrix<R extends ResultRow>`

包：`io.github.sinri.keel.integration.mysql.result.matrix`。

工厂：

- **`ResultMatrix.createSimple(rowSet)`**
- **`ResultMatrix.createSpecific(rowSet, mapper)`**

常用能力：**`getFirstRow` / `getRowByIndex` / `getRowList` / `stream` / `toJsonArray`**；以及 **`buildCategorizedRowsMap`
**、**`buildUniqueKeyBoundRowMap`** 等（唯一键重复会抛 **`IllegalStateException`**）。

**`KeelSQLResultRowIndexError`**：越界或空结果时由 **`getFirstRow`** 等抛出；**`executeForOneRow`** 会捕获并返回 **`null`
**。

## `SimpleResultRow` 与自定义 `ResultRow`

- **`SimpleResultRow`**：继承 **`JsonifiableDataUnitImpl`**，实现 **`ResultRow`**，适合通用 JSON 式读字段（如 *
  *`readIntegerRequired`** 等，以 **`ResultRow`** 接口为准）。
- 自定义行类型：实现 **`ResultRow`** 并提供 **`JsonObject` 构造函数**，以便 **`toMatrix(Class)`** 反射实例化。

## 分页：`PaginationResult`

记录类型：
`io.github.sinri.keel.integration.mysql.result.pagination.PaginationResult(long total, ResultMatrix<?> resultMatrix)`。

- 由 **`RunnableStatementForReadAndPagination#executeForPagination(pageNo, pageSize)`** 生成：**`pageNo` 从 1 开始**，*
  *`pageSize > 0`**。
- **`toJsonObject()`**：包含 **`total`** 与 **`list`**（矩阵转 **`JsonArray`**）。

## 流式处理与 `ResultStreamReader`

接口：`io.github.sinri.keel.integration.mysql.result.stream.ResultStreamReader`。

- 实现 **`Future<Void> read(Row row)`** 处理每一行。
- 静态工具：**`mapRowToEntity(row, clazz)`**（**`row.toJson().mapTo`**）、**`mapRowToResultRow(row, clazz)`**（*
  *`ResultRow.of`**）。

与 **`StreamableStatement`** 配合见 [语句执行与流式读取.md](./语句执行与流式读取.md)。

## 与 `instantQuery` 结果

**`KeelMySQLConfiguration#instantQuery`** 直接返回 **`ResultMatrix<SimpleResultRow>`**，跳过了 **`StatementExecuteResult`
** 这一层；若需要 **`lastInsertId`** 等，应使用连接上的 DML 语句执行路径。
